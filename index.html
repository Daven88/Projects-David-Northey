    Webscraping and NLP for Analysing Autism-Related Book Reviews

    This project applies Natural Language Processing (NLP) and machine learning to analyse book reviews related to Autism.     Forecasting Australia’s Unemployment Rate Using Neural Networks and XGBoost

    This project uses Neural Networks and XGBoost to forecast Australia’s unemployment rate, tackling challenges like limited test data, multicollinearity, and missing values from economic shocks. While XGBoost showed signs of overfitting, Neural Networks offered more stable performance. Future improvements include cross-validation, PCA, and incorporating data from economic downturns to boost model reliability and accuracy.
